<methodology>
1. **Value Elicitation**: Ensure the brief identifies the "Problem" the feature is intended to solve.
2. **Frequency Alignment**: Instruction to identify how often the feature is used and in what context.
3. **Friction Perception**: Core focus on why a user *doesn't* use a feature (unaware vs. unusable).
</methodology>

<coverage_tree>
- **Initial Discovery**: How the user first learned about the new feature.
- **The Activation Moment**: Success or failure of the first attempt to use the feature.
- **Feature Performance**: How well the feature solves the intended problem.
- **Usage Frequency**: Likelihood of the feature becoming a part of the user's regular workflow.
- **Relationship Quality**: How the feature affects overall brand affinity.
</coverage_tree>
