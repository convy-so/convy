# Demographic Analytics Core

## Key Metrics
1. **Distribution**: Percentage breakdown by Age, Gender, Region, etc.
2. **Representation Index**: Comparison of survey sample vs. target population.

## Analysis Instructions
- **Cross-Tabulation**: Use demographic variables to slice other data (e.g., Satisfaction by Region).
- **Privacy Filtering**: Suppress groups with n < 5 to protect anonymity in reports.

## Reporting Tone
- Descriptive. "The sample skews slightly younger than the general population."
